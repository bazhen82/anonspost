AnonsPost — панель email-рассылок
=================================

1. Python 3.11+

2. В папке проекта:
   pip install -r requirements.txt
   copy .env.example .env
   (отредактируйте .env — пароль приложения Mail.ru)

3. Пароль Mail.ru (ОБЯЗАТЕЛЬНО «пароль приложения», не пароль от входа в почту):
   - Откройте https://e.mail.ru/ → Настройки → Безопасность
   - «Пароли для внешних приложений» → Создать → например «AnonsPost»
   - Скопируйте выданный пароль в .env → SMTP_PASSWORD=...
   - SMTP_USER и SMTP_FROM = полный ящик (travel_assistent@mail.ru)
   - Сохраните .env и перезапустите: python app.py

4. Запуск:
   python app.py

5. Браузер: http://127.0.0.1:5000
   Логин/пароль из .env (по умолчанию admin / admin)

6. Порядок:
   - Импорт CSV (пример или свой)
   - Черновик → Сохранить
   - Тест на свой email
   - Запустить рассылку (до 50 писем)

Портфолио-проект, не связан с торговыми марками третьих лиц.
